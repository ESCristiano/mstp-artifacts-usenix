# S world runtime environment

This folder contains the Secure (S) world runtime environment and third-party modules, managed as Git submodules.
- `mbedtls`: Mbed TLS library for the proof-of-concept attacks.
- `mcuboot`: Bootloader providing a Root of Trust (RoT).
- `trusted-firmware-m`: Secure world firmware (TF-M).
- `patches`: Patches enabling additional debugging features in TF-M.
- `tf-m-tests`: Official TF-M tests used as templates for our evaluation.
